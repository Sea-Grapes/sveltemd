# Sveltemd

A modern markdown preprocessor for SvelteKit (WIP). Focused on providing extremely easy markdown usage with minimal config needed.

Goals:

- [x] preprocess markdown files as svelte + markdown
- [ ] flexible layout system (perhaps using a specially-named file that acts like +layout, but only wraps descendants)
- [ ] get frontmatter from children files
- [ ] easy content processing (image, video, audio, etc)

  - relative content urls?
